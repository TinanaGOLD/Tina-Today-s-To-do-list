# 待办清单 (Todo List)
一个功能丰富、界面友好的待办清单网页应用，支持多模式任务管理、计时统计、每日任务等功能，所有数据本地存储，无需注册即可使用。

## 项目介绍
这是一款纯前端实现的待办清单应用，基于原生 HTML + CSS + JavaScript 开发，无需后端依赖。应用不仅包含基础的任务增删改查功能，还扩展了计时模式、每日任务、今日总结等实用功能，界面采用可爱的卡通风格，同时适配移动端和桌面端使用。

## 核心功能
- ✅ 基础任务管理
  - 添加任务：输入任务内容（最多20字）快速创建新待办
  - 编辑任务：支持修改已创建的任务内容
  - 删除任务：移除不需要的任务
  - 标记完成：将完成的任务标记为已完成状态，自动移至列表底部
  - 星标置顶：重要任务可星标置顶，优先展示
- ⏱️ 计时模式
  - 任务计时：对任务进行实时计时，记录完成耗时
  - 暂停/继续：支持计时过程中的暂停和继续操作
  - 耗时统计：完成任务后自动计算并展示总耗时
- 📅 每日模式
  - 每日任务：标记任务为"每日任务"，持久化保存不随模式切换清空
  - 模式切换：独立的每日任务管理空间
- 📊 今日总结
  - 完成统计：自动统计当日完成的任务数量
  - 耗时分析：计时模式下生成任务耗时饼图，直观展示时间分配
  - 数据可视化：饼图展示各任务耗时占比
- 💾 本地存储
  - 自动保存：所有任务数据（包括每日任务、完成状态、计时数据）自动保存至浏览器本地
  - 数据持久：刷新页面/重新打开浏览器后任务数据不丢失

## 技术栈
- **HTML**：构建语义化的页面结构
- **CSS**：实现响应式布局、卡通风格样式及交互动效
- **JavaScript**：实现核心交互逻辑、本地存储、计时功能、数据可视化
